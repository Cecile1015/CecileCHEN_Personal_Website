# Cecile CHEN Academic Website

This repository keeps the same visual template and theme colors as the original `baichen-du.github.io` template, while replacing the content with Cecile CHEN's academic profile.

## Upload to GitHub Pages

1. Create or rename your repository as `Cecile1015.github.io` if your GitHub username is `Cecile1015`.
2. Upload all files in this folder to the repository root.
3. Go to Settings -> Pages.
4. Choose `Deploy from a branch`, then select `main` and `/root`.
5. Visit `https://Cecile1015.github.io` after deployment finishes.

## Replace avatar

Upload your photo as `assets/img/profile.jpg`, then edit `_config.yml`:

```yaml
avatar: ./assets/img/profile.jpg
```

## Replace links

Edit `_config.yml` and replace the Google Scholar / GitHub / ORCID / LinkedIn placeholders with your real profile links.
